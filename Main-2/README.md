# Crypto Portfolio Tracker (Трекер криптовалютного портфеля)

Програмний комплекс призначений для обліку криптовалютних та NFT активів, розрахунку їхньої вартості та експорту фінансових звітів. 

## Основний функціонал
* **Управління активами:** Додавання токенів (`CryptoCoin`) та цифрового мистецтва (`NFT`).
* **Персистентність:** Збереження стану портфеля у бінарний файл (pickle) та експорт звітів у CSV.
* **Fault-Tolerant CLI:** Захищений консольний інтерфейс, який не перериває роботу при помилковому введенні даних.

## Запуск програми
1. Клонуйте репозиторій на свій ПК: `git clone https://github.com/your-username/crypto-tracker.git`
2. Перейдіть у папку проєкту: `cd crypto-tracker`
3. Запустіть головний файл системи: `python main.py`
